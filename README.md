# Detective-Hangman-CMSC255-
Detective Hangman

A Java-based mystery game combining the classic Hangman with the deduction elements of Clue.
Players must solve riddles, guess letters, and uncover clues to determine the motive, weapon, and murderer before running out of attempts.

Game Overview

Detective Hangman is a 3-round interactive word-guessing game where players act as detectives solving a mystery.

How It Works:
- The game displays a sentence with a missing word (a riddle)
- The player guesses one letter at a time
- Each incorrect guess adds a part to the hangman
- Players have 6 total mistakes per round

Game Structure:
Round 1 ----> murderer
Round 2 ----> the victim
Round 3 ----> reveal the motive
Round 4 ----> reveal the location
Round 5 ----> reveal the weapon

Rules
- Maximum 6 incorrect guesses per round
- Hangman resets each round
- If the player fails:
  - The correct word is revealed
  - Game continues to next round
- Final answers determine success
